## Install neovim

```bash
brew install neovim
```

## Download terminal

[release page](https://github.com/kovidgoyal/kitty/releases)

## Config
### Requirement
```bash
brew install ripgrep
brew install fd
```
### Install lazyvim
```bash
git clone https://github.com/LazyVim/starter ~/.config/nvim

rm -rf ~/.config/nvim/.git
```

- start
```bash
nvim
```

### Config keymap (Option)
According to your own preference.
```bash
# ~/.config/nvim/lua/config/keymaps.lua
keymap.set('n', 'jk', '<Esc>')
```


## Shortcut key

- indent `cmd + shift + </>`
- move up/down `A-j/k`
